Arithmetic Sequence Examples

These worked examples show how to identify an arithmetic pattern, find terms and sums, solve for missing information, and translate practical situations into sequence notation.

Quick formula review

For first term a1, common difference d, and position n:

  • Nth term: a(n) = a1 + (n - 1)d
  • Finite sum: S(n) = n[a1 + a(n)] / 2

See the sequence formulas page for a full reference.

Example 1: Find the common difference

Problem: Determine whether 17, 12, 7, 2, -3 is arithmetic.

  1. Subtract consecutive terms: 12 - 17 = -5.
  2. Check another pair: 7 - 12 = -5.
  3. The difference remains -5, so the sequence is arithmetic.

Answer: The common difference is d = -5.

Example 2: Find an nth term

Problem: Find the 15th term of 8, 11, 14, 17, ...

  1. Identify a1 = 8, d = 3, and n = 15.
  2. Substitute: a(15) = 8 + (15 - 1)(3).
  3. Simplify: a(15) = 8 + 42 = 50.

Answer: The 15th term is 50.

Example 3: Calculate a finite sum

Problem: Find the sum of the first 20 terms when a1 = 4 and d = 6.

  1. Find the last term: a(20) = 4 + 19(6) = 118.
  2. Use the sum formula: S(20) = 20(4 + 118) / 2.
  3. Simplify: S(20) = 10(122) = 1220.

Answer: The sum is 1,220.

Example 4: Find a missing first term

Problem: The 12th term is 41 and the common difference is 3. Find the first term.

  1. Start with a(12) = a1 + (12 - 1)(3).
  2. Substitute the known term: 41 = a1 + 33.
  3. Subtract 33: a1 = 8.

Answer: The first term is 8.

Example 5: Find a term position

Problem: In the sequence 5, 12, 19, 26, ..., which term equals 82?

  1. Use 82 = 5 + (n - 1)(7).
  2. Subtract 5: 77 = 7(n - 1).
  3. Divide by 7: 11 = n - 1, so n = 12.

Answer: 82 is the 12th term.

Example 6: A practical savings pattern

Problem: Maya saves $15 in the first week and increases the weekly amount by $4. How much does she save in week 10, and how much has she saved across all 10 weeks?

  1. Use a1 = 15, d = 4, and n = 10.
  2. Week 10: a(10) = 15 + 9(4) = 51.
  3. Total: S(10) = 10(15 + 51) / 2 = 330.

Answer: She saves $51 in week 10 and $330 in total.
