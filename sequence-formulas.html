Sequence Formulas

Use this page as a compact reference for common sequence notation, nth-term formulas, recursive rules, and finite sums. Identify the type of pattern before choosing a formula.

Notation used on this page

SymbolMeaning
a1The first term
a(n)The term at position n
nA positive whole-number position or number of terms
dThe common difference in an arithmetic sequence
rThe common ratio in a geometric sequence
S(n)The sum of the first n terms

Arithmetic sequence formulas

PurposeFormula
Common differenced = a(n) - a(n - 1)
nth terma(n) = a1 + (n - 1)d
Recursive rulea(n) = a(n - 1) + d
Finite sum using last termS(n) = n[a1 + a(n)] / 2
Finite sum using differenceS(n) = n[2a1 + (n - 1)d] / 2

Example: if a1 = 5, d = 3, and n = 8, then a(8) = 5 + 7(3) = 26. The sum is 8(5 + 26) / 2 = 124.

Geometric sequence formulas

PurposeFormula
Common ratior = a(n) / a(n - 1), when the earlier term is nonzero
nth terma(n) = a1 * r^(n - 1)
Recursive rulea(n) = r * a(n - 1)
Finite sum, r not equal to 1S(n) = a1(1 - r^n) / (1 - r)
Finite sum, r = 1S(n) = n * a1
Infinite sum, |r| < 1S = a1 / (1 - r)

Example: if a1 = 4, r = 2, and n = 5, then a(5) = 4 * 2^4 = 64. The first five terms sum to 4(1 - 2^5) / (1 - 2) = 124.

Recursive and Fibonacci formulas

A general recursive formula expresses a term using earlier terms. It must be paired with enough starting values to begin calculation.

  • Arithmetic recursion: a(1) = a1, a(n) = a(n - 1) + d.
  • Geometric recursion: a(1) = a1, a(n) = r * a(n - 1).
  • Fibonacci recursion: F(0) = 0, F(1) = 1, F(n) = F(n - 1) + F(n - 2).

See the recursive sequence guide and Fibonacci sequence guide for step-by-step explanations.

How to choose the right formula

  1. Check subtraction: a constant difference suggests an arithmetic sequence.
  2. Check division: a constant ratio suggests a geometric sequence.
  3. Look for a rule based on earlier terms when neither test is constant.
  4. Determine whether the question asks for one term or a sum.
  5. Write down the known values before substituting them into a formula.

Sequence formula FAQ

Why does the nth-term exponent or multiplier use n - 1?

The first term is already at position 1, so moving from the first term to position n requires n - 1 additions or multiplications.

Can I use the infinite geometric sum formula for any ratio?

No. It applies only when the absolute value of the common ratio is less than 1.

Is there an infinite arithmetic sum formula?

A nonzero constant arithmetic sequence does not have a convergent infinite sum. Arithmetic sum formulas here are finite.

Do sequence positions have to be whole numbers?

In ordinary sequence notation, positions are indexed by whole numbers, commonly beginning at 0 or 1 depending on the definition.
