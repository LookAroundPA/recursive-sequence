Geometric Sequence Examples

These worked examples cover common ratios, nth terms, finite sums, missing values, repeated growth and decay, and sequences with alternating signs.

Quick formula review

For first term a1, common ratio r, and position n:

  • Nth term: a(n) = a1 * r^(n - 1)
  • Finite sum when r != 1: S(n) = a1(1 - r^n) / (1 - r)

See the sequence formulas page for special cases and the infinite sum formula.

Example 1: Find the common ratio

Problem: Determine whether 160, 80, 40, 20, 10 is geometric.

  1. Divide consecutive terms: 80 / 160 = 0.5.
  2. Check another pair: 40 / 80 = 0.5.
  3. The ratio remains 0.5, so the sequence is geometric.

Answer: The common ratio is r = 0.5.

Example 2: Find an nth term

Problem: Find the 7th term of 3, 6, 12, 24, ...

  1. Identify a1 = 3, r = 2, and n = 7.
  2. Substitute: a(7) = 3 * 2^(7 - 1).
  3. Calculate: a(7) = 3 * 64 = 192.

Answer: The 7th term is 192.

Example 3: Calculate a finite sum

Problem: Find the sum of the first 6 terms when a1 = 5 and r = 3.

  1. Use S(6) = 5(1 - 3^6) / (1 - 3).
  2. Calculate 3^6 = 729.
  3. Simplify: S(6) = 5(-728) / (-2) = 1820.

Answer: The sum is 1,820.

Example 4: A negative ratio

Problem: Find the fifth term of 16, -8, 4, -2, ...

  1. The ratio is -8 / 16 = -0.5.
  2. Use a(5) = 16(-0.5)^4.
  3. Because the exponent is even, the result is positive: a(5) = 1.

Answer: The fifth term is 1.

Example 5: Repeated percentage growth

Problem: An amount begins at $800 and grows by 5% each period. Find its value in period 8.

  1. A 5% increase gives the multiplier r = 1 + 0.05 = 1.05.
  2. Use a(8) = 800(1.05)^7.
  3. Calculate and round to cents: a(8) = 1125.68.

Answer: The period-8 value is approximately $1,125.68.

Example 6: Repeated decay

Problem: A measured quantity starts at 240 units and retains 75% of its value each stage. Find the fifth value.

  1. Retaining 75% gives r = 0.75.
  2. Use a(5) = 240(0.75)^4.
  3. Calculate: a(5) = 75.9375.

Answer: The fifth value is 75.9375 units.
