FAQ

What is an Arithmetic Sequence?

An arithmetic sequence is a sequence where the difference between each consecutive term is constant. This constant difference is called the common difference, usually denoted by the letter d.

How to Calculate the nth Term?

The formula for the nth term of an arithmetic sequence is: aₙ = a₁ + (n-1)d, where:
- aₙ is the nth term
- a₁ is the first term
- d is the common difference
- n is the position number

How to Calculate the Sum?

The sum of the first n terms of an arithmetic sequence can be calculated using the formula: Sₙ = n(a₁ + aₙ)/2, where:
- Sₙ is the sum of n terms
- n is the number of terms
- a₁ is the first term
- aₙ is the nth term
